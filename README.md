## 安裝方式

### 安裝前須知

必須要先安裝**Bun**，[Bun官網](https://bun.sh/)。

[![](/readme/bun.png)](https://bun.sh/)

### 安裝指令


```
bun i
```

### 執行指令

```
bun run server.js
```

查看`http://localhost:3000`看到頁面代表完成。

![](/readme/screenshot.png)
